    Methods

    public

    • app() — Returns the global instantiated Strata object.
    • bootstrap() — Bootstraps Strata by creating a Strata instance and saving it to the global scope.
    • config() — Fetches a value in the app's configuration array
    • getBinPath() — Returns the path to the bin folder.
    • getCommandPath() — Returns the path to the project shell command folder.
    • getConfigurationPath() — Returns the path to the configuration folder.
    • getDbPath() — Returns the path to the db folder.
    • getDefaultNamespace() — Returns the default application namespace within Strata
    • getDefaultTestNamespace() — Returns the default namespace for tests within Strata
    • getLocalePath() — Returns the path to Strata's locale configurations.
    • getLogPath() — Returns the path to the log folder.
    • getMUPluginsPath() — Returns the Wordpress must-use plugins path of the project.
    • getMiddlewarePath() — Returns the path to the Middleware folder.
    • getNamespace() — Returns the current project's root namespace key
    • getOurVendorPath() — Returns the path to Strata's vendor folder.
    • getPluginsPath() — Returns the Wordpress plugins path of the project.
    • getProjectConfigurationFilePath() — Returns the path to Strata's configuration file.
    • getRootPath() — Returns the root path of the project.
    • getSRCPath() — Returns the path to the src folder.
    • getShellPath() — Returns the path to the project shell folder.
    • getTestPath() — Returns the path to the test folder.
    • getThemesPath() — Returns the Wordpress themes path of the project.
    • getTmpPath() — Returns the path to the temporary folder.
    • getUtilityPath() — Returns the path to the utility folder.
    • getVendorPath() — Returns the path to the project's vendor folder.
    • getWordpressPath() — Returns the Wordpress path.
    • i18n() — Returns the current localization object of the instantiated Strata object.
    • isBundledServer() — Returns whether Strata is running from the command line server
    • isCommandLineInterface() — Returns whether Strata is running from the CLI tool but not the bundled server.
    • isDev() — Returns whether Strata is running in development mode.
    • isTest() — Returns whether Strata is running in test mode.
    • parseProjectConfigFile() — Loads ~/config/strata.php, cleans up the data and saves it as the active configuration of the current instance of the Strata object.
    • requireVendorAutoload() — Includes Composer's generator autoloader from the vendor directory
    • rewriter() — Returns the current URL rewriter of the instantiated Strata object.
    • router() — Returns the current URL router of the instantiated Strata object.
    • writeProjectConfigFile() — Write the configuration object to the project's configuration file.
