Strata\Shell\Command\Generator\GeneratorBase

Automates repetitive creation of code files. It validates the class names and file locations based on the set of guidelines promoted by Strata.

Intended use include:
<code>./strata generate controller User</code>
<code>./strata generate customposttype Fruit</code>
...
